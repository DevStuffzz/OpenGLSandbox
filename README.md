# OpenGLTemplate
An OpenGL Template for C++ and visual studio 


# Usage
1. use ```git clone --recursive https://github.com/DevStuffzz/OpenGLTemplate/edit/main/README.md```
2. Open up the project and run "scripts/Win-GenerateProjects.bat" if you are not using VS2019 open the batch file and change the target to your vs version
3. Open The Solution and Run
4. Make any changes you want to the project name by changing them in the correct premake5 files
5. These files are the one in the root directory and in "OpenGLApp" note that you may need to rename this directory
